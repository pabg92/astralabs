# ContractBuddy

AI-powered contract reconciliation platform for influencer marketing agencies.

---

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- **Frontend:** Next.js 15, React 19, TypeScript 5, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL), pgvector, pgmq
- **AI:** OpenAI GPT-4o, Cohere embeddings
- **Auth:** Clerk (planned)
